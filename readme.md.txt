# Pauline Dapash's Portfolio

![Website Preview](images/maasai-bg.jpg)

A cultural portfolio showcasing Pauline Dapash's web development skills and Maasai culinary heritage.

## Features

- Interactive profile with traditional attire toggle
- Authentic Maasai-inspired recipes
- Animated skill bars
- Responsive design
- Maasai color scheme and patterns

## Pages

1. **Home** - Introduction and featured recipes
2. **About** - Interactive profile with cultural background
3. **Ugali & Fish** - Maasai-style recipe
4. **Chapati & Beans** - Pastoralist-inspired recipe

## Setup

1. Clone repository
2. Add these images to `/images`:
   - `maasai-bg.jpg`
   - `profile.jpg`
   - `traditional.jpg`
   - `ugali-fish.jpg`
   - `chapati.jpg`
3. Open `index.html` in browser
